# 🚍 Catchbus  
차고지·회차지 출발시각 예측 기반 실시간 버스 도착 안내 애플리케이션

> **2022-2 파란학기제 Catchbus 프로젝트**  
> 카카오/네이버 지도에서 실시간 버스 도착 서비스를 제공하기 이전에 개발된 실시간 버스 정보 서비스입니다.

---

## 📌 개요
- 공공데이터포털의 버스 도착 정보 API를 기반으로 실시간 버스 도착 정보를 제공합니다.  
- 차고지·회차지에서의 **도착 예측이 어려운 문제**를 해결하기 위해  
  직접 **Rule-Based 알고리즘을 설계**하여 예측값을 보정했습니다.

---

## 🛠 주요 기능
- 실시간 버스 도착 정보 제공  
- 특정 버스/정류장 검색  
- 공공데이터 API 연동  
- 차고지/회차 구간 예측 알고리즘 적용 -> 앱 push 제공
- 대중교통-기차-시외버스 통합 end-to-end 여행 지원
